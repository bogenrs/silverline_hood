# Silverline Hood Integration

Control your Silverline range hood directly through Home Assistant!

## Features

✅ **Fan Control** - 4 speed levels  
✅ **RGBW Lighting** - Full color control  
✅ **Bidirectional Sync** - Detects remote control changes  
✅ **Configurable Poll Interval** - 5-300 seconds  

## Installation

1. Install the integration via HACS
2. Restart Home Assistant  
3. Add the integration through the UI
4. Enter the IP address of your range hood

## Configuration

After installation, you can configure the poll interval via:
**Settings** → **Devices & Services** → **Silverline Hood** → **Configure**

## Supported Devices

- Silverline range hoods with Telnet support (Port 23)
- JSON command structure with motor and light control

## Silverline Hood Integration

Steuern Sie Ihre Silverline Dunstabzugshaube direkt über Home Assistant!

## Features

✅ **Lüfter-Steuerung** - 4 Geschwindigkeitsstufen  
✅ **RGBW-Beleuchtung** - Vollständige Farbsteuerung  
✅ **Bidirektionale Synchronisation** - Erkennt Fernbedienungs-Änderungen

## Installation

1. Installieren Sie die Integration über HACS
2. Starten Sie Home Assistant neu  
3. Fügen Sie die Integration über die UI hinzu
4. Geben Sie die IP-Adresse Ihrer Dunstabzugshaube ein

## Konfiguration

Nach der Installation können Sie das Abfrageintervall über:
**Einstellungen** → **Geräte & Dienste** → **Silverline Hood** → **Konfigurieren**

## Unterstützte Geräte

- Silverline Dunstabzugshauben mit Telnet-Unterstützung (Port 23)
- JSON-Befehlsstruktur mit Motor- und Lichtsteuerung
